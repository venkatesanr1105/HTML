WEB STORAGE

This example is designed to demostrate usage of the W3C Web Storage API. It should wok as far back as IE8. Choose custom background colours, logos and fonts from the below drop down menus, then try closing and reopenning teh page - you will find that your choice are remembered, as they are stored using Web Storage. You can also visit the storage event output (open in new tab). Open this, change some values in the index page, then look at the events page - you'll see the storage changes reported.
